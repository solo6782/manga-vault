v1.0.0 Mars 2026

Fonctionnalités

  • Authentification Supabase (inscription / connexion / déconnexion)
  • Ajout manga et anime avec recherche Jikan API (MyAnimeList)
  • Auto-remplissage : titre, auteur/studio, année, format, genres, cover, score MAL
  • Titres anglais affichés en sous-titre dans les résultats de recherche
  • Wizard étape par étape lors de l'ajout (statut → progression → note → commentaire)
  • Modal de complétion quand une œuvre passe à "Terminé"
  • Système de notation 1-10 étoiles
  • Genres avec tags cliquables
  • Filtres : type (manga/anime), statut, recherche textuelle
  • Tri par titre, année, note
  • Groupement par univers/franchise avec crawl récursif des relations Jikan
  • Modale univers : œuvres possédées + exploration complète
  • Liens MAL sur les cartes (score cliquable) et dans la modale univers
  • Sous-titres anglais et année dans la modale univers
  • Mise à jour automatique des scores MAL au chargement
  • Badges de statut avec couleurs contrastées
  • Interface responsive et dark mode

Architecture

  • Frontend : Vanilla JS / HTML / CSS
  • Backend : Supabase (PostgreSQL + Auth)
  • Hosting : Cloudflare Workers
  • API : Jikan v4 (MyAnimeList)
